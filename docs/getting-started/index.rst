Getting Started
===============

This guide will help you install sphinx-typesense and configure your first
Typesense-powered documentation search.

.. toctree::
   :maxdepth: 2

   installation
   quickstart

Prerequisites
-------------

Before you begin, you will need:

1. **Python 3.9+** - sphinx-typesense requires Python 3.9 or later
2. **Sphinx 7.0+** - A Sphinx documentation project
3. **Typesense Server** - Either:

   - A self-hosted Typesense server (see `Typesense Installation Guide <https://typesense.org/docs/guide/install-typesense.html>`_)
   - A Typesense Cloud account (see `Typesense Cloud <https://cloud.typesense.org/>`_)

What You Will Learn
-------------------

By the end of this guide, you will:

- Have sphinx-typesense installed in your project
- Understand the basic configuration options
- Have a working search integration in your documentation

Time Required
-------------

- **Installation**: 2 minutes
- **Basic Configuration**: 5 minutes
- **First Build with Search**: 5 minutes

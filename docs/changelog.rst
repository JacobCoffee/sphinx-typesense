[0.1.0] - 2025-01-19
--------------------

Initial release.

Added
~~~~~

- Sphinx extension with Typesense search integration
- Build-time content indexing with DocSearch-compatible schema
- Support for multiple Sphinx themes (RTD, Furo, Alabaster, PyData, Book)
- Configurable typo tolerance (0-2 typos)
- Environment variable support for API keys
- Search UI injection with JavaScript configuration
- CSP-compliant meta tag configuration option
- Comprehensive documentation
